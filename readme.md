# 📚 Relational Database (SQL & MySQL)

## 📌 What is a Database?

A **database** is a place where we store data in an organized way.

👉 Example:

* Student records
* Employee details
* Product information

Instead of storing data in files, we use databases to:

* Store data safely
* Access data easily
* Manage large amounts of data

---

## 📌 Types of Databases

There are mainly **2 types of databases**:

### 1. Relational Database (RDBMS)

* Data is stored in **tables (rows & columns)**
* Tables are related to each other
* Uses **SQL (Structured Query Language)**

👉 Example:

* MySQL
* PostgreSQL
* Oracle

---

### 2. Non-Relational Database (NoSQL)

* Data is stored in **different formats** (not tables)
* Used for large-scale and flexible data

👉 Types:

* Document-based (MongoDB)
* Key-Value (Redis)
* Graph (Neo4j)

---


## 📌 What is SQL?

**SQL (Structured Query Language)** is used to:

* Create databases
* Insert data
* Update data
* Delete data
* Retrieve data

👉 Example:

```sql
SELECT * FROM student;
```

---

## 📌 What is MySQL?

**MySQL** is a **Relational Database Management System (RDBMS)**.

👉 It uses SQL to manage data.

Features:

* Open-source
* Fast and reliable
* Easy to use
* Widely used in web development

---

## 📌 Other SQL Tools / Databases

* PostgreSQL
* Oracle Database
* Microsoft SQL Server
* SQLite

👉 SQL is common, but each tool has some differences.

---

## 📌 Why SQL is Important?

SQL is very important because:

* 🔹 Used in almost every application
* 🔹 Handles large data easily
* 🔹 Helps in data analysis
* 🔹 Required for backend development
* 🔹 Used in companies for data storage

---


## 📌 Advantages of Relational Database

* Data is structured
* Easy to understand
* Data integrity (accuracy)
* Relationships between tables

---

## 📌 Disadvantages

* Not flexible for unstructured data
* Can be slow for very large data
* Scaling is difficult

---

## 🚀 Conclusion

* Database helps store and manage data
* Relational databases use tables
* SQL is used to work with data
* MySQL is one of the most popular databases
